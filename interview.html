SDE / SWE

What are SOLID principles?
SOLID stands for five key OOP principles: SRP, OCP, LSP, ISP, DIP.
Design a URL shortener.
Use hashing, key-value storage, and redirection logic to map short codes to full URLs.
Difference between stack and queue?
Stack is LIFO (last in, first out); queue is FIFO (first in, first out).
How do you debug a production issue?
Use logs, breakpoints, and binary search to isolate and resolve the issue efficiently.

Data Analyst

How do you handle missing data?
Use imputation, deletion, or predictive modeling depending on context and data type.
Inner vs outer join?
Inner joins return matched rows; outer joins include unmatched rows from one or both tables.
How to visualize trends over time?
Use line charts, area charts, or time series plots to show patterns and changes.
Describe a data-driven decision project.
Showcase how your analysis led to measurable business impact or strategic change.

Data Scientist

What is overfitting?
Overfitting means the model memorizes noise; prevent it with regularization or more data.
Bias-variance tradeoff?
Bias is error from assumptions; variance is error from sensitivity to data fluctuations.
How to evaluate classification models?
Use metrics like accuracy, precision, recall, F1-score, and ROC-AUC depending on the context.
Metrics for recommendation systems?
Use precision@k, recall@k, NDCG, and mean reciprocal rank to evaluate recommendations.

DevOps

What is DevOps?
DevOps integrates development and operations to improve collaboration and accelerate delivery.
Explain CI/CD pipeline.
CI/CD automates code integration, testing, and deployment for faster and safer releases.
Docker vs Kubernetes?
Docker packages applications into containers; Kubernetes manages and scales those containers.
What is shift-left testing?
Shift-left testing means testing earlier in the development cycle to catch issues sooner.

UI/UX Designer

Difference between UI and UX?
UI focuses on visual design and layout; UX focuses on user experience and interaction flow.
Wireframe vs mockup?
Wireframes are structural blueprints; mockups are detailed visual representations of the interface.
How do you improve UX?
Conduct user research, usability testing, and iterate designs based on feedback and analytics.
Why is accessibility important?
Accessibility ensures that all users, including those with disabilities, can use your product.

Product Manager

What does a PM do?
A PM defines product vision, prioritizes features, and aligns teams to deliver user value.
How do you prioritize features?
Use frameworks like RICE, MoSCoW, or Kano to balance impact, effort, and urgency.
Key metrics for product success?
Track retention, engagement, NPS, conversion rate, and revenue growth.
How do you handle trade-offs?
Balance user needs, technical feasibility, and business goals through stakeholder alignment.

QA Engineer

What is smoke testing?
Smoke testing checks basic functionality before deeper testing begins.
What is regression testing?
Regression testing ensures new changes don‚Äôt break existing features.
Why use automation in QA?
Automation speeds up testing, improves coverage, and reduces human error.
How do you report bugs?
Use bug tracking tools with clear steps, severity, screenshots, and expected vs actual behavior.
